vCard Generator

A simple, open source and self hosted vCard (virtual contact card) generator web app that helps users create, preview, and download vCard (.vcf) files with contact details and a profile photo. Modern and supports Google Contacts, iOS Contacts and Outlook Contacts. Built with Alpine.js, Tailwind CSS, and a lightweight PHP for vCard generation.
